# *2Think*

`cuadrado` y `resumir` tienen estructura idéntica: crean un array del mismo tamaño que el original, recorren el original con un bucle y escriben en cada posición el resultado de una transformación.

La única diferencia es qué transformación aplican.

¿Qué tendría que cambiar en el diseño para que ambas fueran la misma función con distinta transformación como parámetro?
